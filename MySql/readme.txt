

Entorno de Desarrollo:NETBEANS
LENGUAJE: jAVA 

1-Ejecutar el archivo en un IDE que soporte el lenguaje JAVA ( preferentemente NETBEANS)

2- Al ejecutar el programa , se abrira un panel donde se debera introducir los Datos correspondientes( primero los de la farmacia , luego
los de usuario)

3- El programa de acuerdo con la opcion seleciionada realizara lo siguiente: 

-Guardar:cargara los datos en una Base de Datos
-Consultar: Mostrara los Datos ya cargados 
- En la seccion de Datos del usuario:Farma cercana: Mostrara el Listado de Farmacias con la Distancia que el usuario debe recorrer
de acuerdo a la Longitud ingresada ( tanto de Farmacia como de usuario ).                                                                                                       

NOTA: CADA CLASE CUENTA CON UNA BREVE INTRODUCCION DE LA FUNCION QUE CUMPLE.
